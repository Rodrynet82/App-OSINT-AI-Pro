# OSINT AI Pro 🚨 **V3.0 - Globe.gl Edition**

Plataforma web de **inteligencia de amenazas OSINT** con IA avanzada. Dashboard profesional con **globo 3D interactivo** (Globe.gl), 19 pruebas automatizadas, reportes PDF/JSON y suite de herramientas expandida.

[![Demo en Vercel](https://app-osint-ai-pro.vercel.app)

## ✨ **Características destacadas**

### **Dashboard Interactivo**
- **Métricas clicables** (Análisis IA: 147, Investigaciones: 3,247, Score: 67, Países: 89).
- Modales con **gráficos Chart.js**: línea, donut, barras horizontales, tablas interactivas.
- **Globo 3D de amenazas** con arcos láser, rings de impacto y rotación automática.
- Feed de amenazas en vivo con eventos clickables y filtros.

### **Investigación Inteligente**
- Score de riesgo con veredicto IA.
- Geolocalización simulada y huella digital OSINT.
- Exportación JSON/PDF inmediata.

### **Suite de Herramientas OSINT (40+)**
Categorías: **Red/IP**, **Email/Domain**, **Social Media**, **Análisis Forense**.
- Toggles para activar/desactivar.
- Panel inline de ejecución con resultados en tiempo real.
- Ejemplos: WHOIS, Port Scanner, Breach Hunter, Username Search, Hash Analyzer.

### **Reportes & Monitoreo**
- Generador PDF/JSON con jsPDF + html2canvas.
- Monitoreo persistente (localStorage) con alertas automáticas.
- Configuración avanzada de APIs y notificaciones.

## 🛠️ **Stack Técnico**
- Frontend: HTML5 - Vanilla JS - CSS3 (CSS Variables)
- Librerías: Globe.gl (mapa 3D) - Chart.js (gráficos) - jsPDF + html2canvas (PDF)
- Estilos: Font Awesome 6 - Design Tokens - Animaciones CSS
- Deploy: Vercel (estático)
